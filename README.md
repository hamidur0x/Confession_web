## 💻 How to Run Locally

1. 📥 Download or clone the repository.  
2. 📂 Make sure all files are in the same directory.  
3. 🌟 Open `index.html` in your web browser **or** use VS Code Live Server extension to run the site locally.  
4. 🎉 Enjoy the website!
